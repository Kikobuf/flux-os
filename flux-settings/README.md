# Flux Settings

GUI settings panel built with Tauri + React.
Replaces all vim-based config editing with a visual interface.

## Tabs
- Appearance (dark/light, accent color, wallpaper, fonts)
- Display (monitor arrangement, scaling, refresh rate)
- Network (WiFi, VPN client UI, speed indicator toggle)
- Security (firewall, auto-updates, encryption)
- Performance (CPU/RAM taskbar toggle, battery mode)
- Startup Apps
- Updates
- About
