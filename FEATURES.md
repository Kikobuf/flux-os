# Flux Features

## OS & UX
- GUI settings panel (replaces all vim-based config editing)
- Taskbar mode option (alongside tiling mode)
- Right-click desktop menu
- Dark / light mode toggle
- Accent color system (one color propagates system-wide)
- Wallpaper picker GUI
- Font picker GUI
- Onboarding wizard (first-boot setup flow)
- Startup app manager GUI
- Screenshot tool GUI (Snipping Tool equivalent)
- Crash reporter

## AI
- AI setup assistant (Claude-powered, conversational system configuration)
- All config editing done through GUI — no vim, ever

## Hardware
- Automatic GPU driver detection (NVIDIA / AMD / Intel)
- NVIDIA out-of-the-box compatibility fixes
- Auto monitor detection wizard (plug in monitor → GUI appears)
- Touchpad gestures (swipe to switch workspaces, pinch to zoom)
- Battery optimization mode toggle (laptop users)

## Files
- Dolphin file manager (replaces Nautilus)
- Trash that always works (right-click → delete goes to trash)
- Recent files (in file manager and app launcher)

## Network & Security
- VPN client UI (bring your own WireGuard / OpenVPN config)
- Firewall GUI (simple on/off with app rules)
- Auto-updates toggle (automatic vs manual)
- Full disk encryption (clear setup during OOBE)
- Network speed indicator in taskbar (toggle in settings)

## Performance
- CPU / RAM usage in taskbar (toggle in settings)
- SSD TRIM scheduler (automatic, silent)

## Connectivity
- LocalSend (AirDrop-style file sharing, cross-platform)
- KDE Connect (phone ↔ desktop: clipboard, notifications, files)
