# Flux

**Linux, reimagined for the AI age.**

Flux is a beautiful, beginner-friendly, AI-first Linux distribution based on [Arch Linux](https://archlinux.org/) and [Hyprland](https://hypr.land/). It's a fork of [Omarchy](https://omarchy.org) by DHH, rebuilt for developers and AI builders who want the power of Linux without the pain.

## What makes Flux different?

- **GUI Settings Panel** — no vim configs, ever. Change everything visually
- **AI Setup Assistant** — Claude-powered onboarding that configures your system through conversation
- **Windows-feel UX** — taskbar mode, right-click desktop menu, accent colors, dark/light toggle
- **AI-first app stack** — Claude Code, OpenCode, Ollama, LM Studio, Open WebUI, Antigravity out of the box
- **Beginner-proof hardware** — automatic GPU detection, NVIDIA fixes, monitor wizard, touchpad gestures
- **Visual App Installer** — browse and install apps from a grid UI, not a terminal menu

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/Kikobuf/flux-os/main/boot.sh | bash
```

> Requires a dedicated drive. Full disk encryption enabled by default.
